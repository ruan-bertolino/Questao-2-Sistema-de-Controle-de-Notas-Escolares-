# Questao-2-Sistema-de-Controle-de-Notas-Escolares-
Sistema de gerenciamento de notas em Java para a turma "3º Ano A". O projeto cadastra alunos, calcula médias e permite interações via menu interativo com opções para emitir boletins, contar aprovados, exibir o aluno com maior média e lançar novas notas.
